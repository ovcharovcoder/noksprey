Збірка gulp (проста)
Автор: Андрій Овчаров
Дата створення: 18.11.2022


Можливості збірки:
- конвертує scss в css;
- стискає файли css, js та зображення;
- розробка проекту виконується у теці "app";
- готовий проект конвертується у теку "dist".


Команди для роботи зі збіркою:

npm i (розпакування збіру)

gulp (Запустити режим розробки)

gulp build (Запустити режим збірки для продакшина)
